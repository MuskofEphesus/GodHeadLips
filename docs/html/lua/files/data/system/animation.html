Functions

Object.animate (self, args)
Sets or clears an animation.

Parameters

  • self: Object.
  • args: Arguments.
    • animation: Animation name.
    • channel: Channel number.
    • fade_in: Fade in duration in seconds.
    • fade_out: Fade out duration in seconds.
    • node_weight: Sets blending weight multipliers for individual nodes.
    • weight: Blending weight.
    • time: Starting time.
    • time_scale: Time scaling factor.
    • permanent: True if should keep repeating.
  • repeat_start: Starting time when repeating.

Return value:

True if started a new animation.

Fades out an animation channel.

Parameters

  • self: Object.
  • args: Arguments.
    • channel: Channel number.
    • duration: Fade duration in seconds.
Object.edit_pose (self, args)
Edits the pose of a node.

Parameters

  • self: Object.
  • args: Arguments.
    • channel: Channel number.
    • frame: Frame number.
    • node: Node name.
    • position: Position change relative to rest pose.
    • rotation: Rotation change relative to rest pose.
    • scale: Scale factor.
Object.find_node (self, args)
Finds a bone or an anchor by name.

Parameters

  • self: Object.
  • args: Arguments.
    • name: Node name.
    • space: Coordinate space. ("local"/"world")

Return value:

Position and rotation, or nil if not found.
Object.get_animation (self, args)
Gets animation information for the given animation channel.
If an animation is looping in the channel, a table containing the fields animation, time, and weight is returned.

Parameters

  • args: Arguments.
    • channel: Channel number. (required)

Return value:

Animation info table or nil.
