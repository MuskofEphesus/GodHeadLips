Quaternion.__mul (self, value)
Multiplies the quaternion with another value.
The second value can be a scalar, a vector, or another quaternion. If it is a scalar, all the components of the quaternion are multiplied by it. If it is a vector, the vector is rotated by the quaternion, and if it is another quaternion, the rotations of the quaternions are concatenated.

Parameters

  • self: Quaternion.
  • value: Quaternion, vector, or number.
