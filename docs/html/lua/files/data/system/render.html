Scene.render (self, args)
Renders the scene.

Parameters

  • self: Scene.
  • args: Arguments.
    • hdr: True to enable HDR.
    • modelview: Modelview matrix.
    • multisamples: Number of multisamples. Pass arguments:
    • postproc_passes: Array of post-processing passes.
      • mipmaps: True to generate mipmaps.
      • shader: Post-processing shader name.
    • projection: Projection matrix.
    • render_passes: Array of render passes. Pass arguments:
      • pass: Pass number.
      • sorting: True to enable sorting.
    • viewport: Viewport array.
